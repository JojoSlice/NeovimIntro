Vim Modes Guide

En översikt över de olika lägena i Vim och Neovim

1. Normal Mode

I Normal Mode navigerar och manipulerar du text utan att skriva direkt.

  • h, j, k, l - Navigera (vänster, ner, upp, höger)
  • dw - Radera ett ord
  • w, b - Hoppa till nästa/föregående ord

2. Insert Mode

I Insert Mode skriver du direkt i texten, som i en vanlig editor.

  • i - Infoga text vid markören
  • a - Börja skriva efter markören
  • Esc - Gå tillbaka till Normal Mode

3. Visual Mode

Visual Mode används för att markera text för redigering eller kopiering.

  • v - Markera tecken
  • V - Markera hela rader
  • Ctrl-v - Markera block

4. Command-Line Mode

I Command-Line Mode skriver du kommandon för att spara, avsluta eller söka.

  • :w - Spara filen
  • :q - Avsluta
  • :wq - Spara och avsluta

5. Replace Mode

Replace Mode låter dig ersätta tecken direkt utan att behöva radera först.

  • r - Ersätt ett enskilt tecken
  • R - Gå in i Replace Mode för att skriva över text
